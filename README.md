# SketchFlow (Vercel-ready)

Modern responsive drawing web app with:
- Auth page
- Main drawing app
- Saved drawings history
- Delete drawings

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Import project in Vercel.
3. Framework preset: **Next.js** (auto-detected).
4. Click **Deploy**.

No environment variables needed.
